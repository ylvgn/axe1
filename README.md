## How to Build

### Build on Windows

#### Requirements
- Visual Studio 2022
- Windows SDK 10.0.19041.0
```
install_vcpkg_packages.bat
gen_vs2022.bat
```
- Open Visual Studio project in `_build/axe1-x64-windows`
